## Node.js를 소개합니다.

React.js는 Node.js를 기반으로 동작

JavaScript 실행 환경 => 구동기

## Node.js 사용하기

### 패키지
- Node.js에서 사용하는 프로그램의 단위

package.json의 scripts 부분은 매크로를 생성하는 부분!

"start" : "node src/index.js"

이렇게 적혀있으면 
`npm run start` 이 명령어를 터미널에 실행 시 
적혀있는 코드(node src/index.js)가 실행됨!
